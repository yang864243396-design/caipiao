## 19. 安全与越权

| # | 步骤 | 预期 |
|---|------|------|
| 19.1 | 无 token 调 `/client/*` | 401 |
| 19.2 | Client token 调 `/admin/*` | 403 |
| 19.3 | A 会员 token 查 B 会员数据（改 memberNo/instanceId） | 拒绝/查不到，**无越权** |
| 19.4 | 篡改/过期 JWT | 401，不泄露信息 |
| 19.5 | 解绑/激活他人 guaji 账号 id | 仅能操作自己绑定的（accountsvc 带 member_id 校验） |
| 19.6 | Admin RBAC：低权限角色调高权限 API | 403 / 菜单不可见 |
| 19.7 | 第三方密码/Token 是否泄露 | 任何 API 响应/日志**不含**明文密码/Token（仅加密存储） |
| 19.8 | SQL 注入（订单号/搜索框输入 `' OR 1=1`） | 参数化查询，无注入 |
